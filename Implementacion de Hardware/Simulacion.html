En la siguiente simulación se presenta un proyecto de Implementación de Hardware. Es una Wireless Sensor Network con dos actores principales los cuales son el Estante Inteligente encargado de administrar los componentes que están a disposición del Brazo Robot. El otro actor principal es el Brazo Robot encargado de ensamblar autos y sus partes, pintar autos, determinar los componentes necesarios para un ensamble y además es capaz de manipular las piezas disponibles.

  • Etiquetas RFID
  • Lector de RFID
  • Obtener Siguiente Componente del Sistema
  • Sensor Verificador de Componentes Instalados
  • Sensor Verificador de Componentes Pintados/Estilizados
  • Sensor medición de Componentes
